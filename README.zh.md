# e-commerce-skills — Claude Code / Codex 电商技能包

本文件与 `README.md` 保持同一中文说明入口。

`e-commerce-skills` 用于沉淀可复用的电商工作流，包括商品策略、卖点提炼、详情页策划、货架内容优化、直播带货脚本、达人种草 brief、大促策略和活动复盘。

## 快速开始

```text
/plugin marketplace add saibodafu/e-commerce-skills
/plugin install e-commerce-skills
```

或在 Codex 中打开本仓库后直接使用：

```text
使用 e-commerce-skills，帮我整理一个电商工作流。
```
